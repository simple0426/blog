Pretty JSON

按Ctrl+Alt+J就可格式化json数据

Table Editor

  • 通过Ctrl+Shift+P->Table Editor: Enable for current view开启
  • 先输入标题行,如|id|name|age|，回车后在第二行输入|-后，
    按tab键就将进入table编辑模式
  • 表格必须与前面输入的文字之间有空行，否则表格会被当成普通文字渲染

Markdown Editing

  • 设置：package setting–>markdown editing–>GFM setings User
