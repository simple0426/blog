# 进程管理
* free：显示内存【内存状态】
    - -m：以M为单位
    - -h：以友好的单位
* uptime：显示系统负载

# 磁盘与文件管理
* iostat【磁盘状态】
    - tps：每秒钟收到的io请求数
    - kB_read/s：每秒钟读的字节数
    - kB_wrtn/s：每秒钟写的字节数
    - kB_read：读取字节总数
    - kB_wrtn：写字节总数
